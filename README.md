# 자웃자 (Jautja) - 프리미엄 오디오 쇼핑몰

휴대용 방송 장비 전문 프리미엄 쇼핑몰 웹사이트입니다.

## 실행 방법

1. **의존성 설치**
   ```bash
   npm install
   ```

2. **개발 서버 실행**
   ```bash
   npm run dev
   ```

3. 브라우저에서 http://localhost:3000 접속

## 빌드

```bash
npm run build
```

빌드 결과물은 `dist` 폴더에 생성됩니다.

## 주요 기능

- **헤더**: 브랜드 로고, 네비게이션, 검색/장바구니/관리자 버튼
- **히어로 섹션**: 메인 배너와 CTA 버튼
- **대표 상품**: 시그니처 제품 상세 소개
- **상품 갤러리**: 전체 상품 그리드
- **배너 섹션**: 추가 프로모션 영역
- **블로그 섹션**: 스튜디오 팁 & 리뷰
- **푸터**: 연락처, 링크, 뉴스레터 구독
- **관리자 대시보드**: 우측 상단 사용자 아이콘 클릭 시 테마 설정 및 상품 CMS

## 기술 스택

- React 19 + TypeScript
- Vite
- Tailwind CSS (CDN)
- Lucide React (아이콘)
